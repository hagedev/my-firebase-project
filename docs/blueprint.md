# **App Name**: AirCafe

## Core Features:

- Tampilan Menu: Menampilkan menu kafe, termasuk kategori dan status stok real-time, di PWA.
- Pemesanan: Memungkinkan pelanggan melakukan pemesanan melalui PWA dengan pemilihan varian dan verifikasi anti-spam.
- Pembayaran QRIS: Mendukung pembayaran QRIS dengan menambahkan kode unik 3 digit ke total tagihan untuk verifikasi pembayaran.
- Pelacakan Pesanan: Memungkinkan pelanggan melacak status real-time pesanan mereka (Diterima -> Disiapkan -> Siap Diantar).
- Manajemen Pesanan: Memungkinkan pengguna admin menerima notifikasi real-time untuk pesanan baru, memverifikasi pembayaran, dan mengelola status pesanan.
- Manajemen Menu & Stok: Memungkinkan admin melakukan operasi CRUD untuk item menu, kategori, dan varian, dengan tombol toggle untuk status stok.
- Pembuatan Token Harian: Memungkinkan administrator kafe membuat token harian untuk dibagikan kepada pelanggan untuk tujuan otentikasi.

## Style Guidelines:

- Warna primer: Oranye hangat (#FF9800) untuk membangkitkan perasaan hangat, energi, dan nafsu makan.
- Warna latar belakang: Oranye pucat muda (#FFF3E0), memberikan latar belakang yang lembut dan mengundang.
- Warna aksen: Kuning keemasan (#FFC107), digunakan hemat untuk elemen interaktif dan notifikasi untuk menarik perhatian pengguna.
- Font judul: 'Playfair', serif, untuk tampilan modern dan elegan. Font Isi: 'PT Sans', sans-serif.
- Gunakan ikon datar dan minimalis berwarna oranye dan kuning keemasan untuk mewakili kategori menu dan status pesanan.
- Terapkan tata letak yang bersih dan modern dengan fokus pada desain mobile-first untuk PWA konsumen, dan desain responsif untuk aplikasi web admin.
- Masukkan animasi halus untuk pembaruan status pesanan dan transisi menu untuk meningkatkan pengalaman pengguna.
